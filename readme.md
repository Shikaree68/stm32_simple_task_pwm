### Описание реализации тестового задания

## Задание

	На любой плате STM32 в Cube\IDE создать проект, который:
	1. создает виртуальный ком-порт на usb для доступа с компьютера
	2. активируется таймер с выходным PWM (на произвольной ножке по выбору)
	3. по АТ-команде ATNxxxx (например ATN1000) устанавливается ширина PWM-импульса в xxxx микросекунд, что можно будет увидеть осциллографом
	4. по нажатию пользовательской кнопки ключает\выключает таймер PWM
	
## Про проект

	Проект был реализован для STM32F103C8T6 "BluePill".
	Реализована работа кнопки в неблокирующем режиме.
	Проект протестирован.
	
## Сборка

	Склонировать проект. Запустить открыть файл .ioc и перегенерировать проект. После будет возможна его сборка через CubeIde.
